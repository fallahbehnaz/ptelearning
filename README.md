# اپ آموزش زبان انگلیسی 📱

یه اپلیکیشن ساده برای یادگیری زبان انگلیسی (کالوکیشن + لغات تکی) با فلش‌کارت، آزمون و تلفظ صوتی، ساخته‌شده با React Native و Expo.

## امکانات
- دو بخش کالوکیشن و لغات با فلش‌کارت و آزمون
- بخش «اعداد»: تمرین خوانش اعداد با گوش دادن (چندگزینه‌ای یا تایپی)
- **بررسی خودکار تلفظ** توی بخش اعداد: بچه عدد رو با صدای خودش می‌گه، اپ گوش می‌ده و می‌گه درست گفته یا نه
- تلفظ صوتی (Text to Speech) روی همه‌ی بخش‌ها
- ذخیره پیشرفت روی گوشی، جدا برای هر بخش

## نصب و اجرا (توجه: این پروژه دیگه با Expo Go ساده اجرا نمی‌شه)

از وقتی «بررسی تلفظ» اضافه شده، اپ به یه ماژول نیتیو به اسم `expo-speech-recognition` نیاز داره که **توی اپ Expo Go وجود نداره**. برای همین باید یه نسخه‌ی اختصاصی (Development Build) از اپ بسازی. این کار یه بار انجام می‌شه و بعدش مثل قبل کار می‌کنی.

### پیش‌نیازها
1. [Node.js](https://nodejs.org) نسخه ۱۸ به بالا
2. یه حساب رایگان روی [expo.dev](https://expo.dev) (برای ساخت بیلد ابری، نیاز به نصب Android Studio نیست)

### مرحله ۱: نصب پکیج‌ها
```bash
cd english-app
npm install
npx expo install expo-speech
npm install expo-speech-recognition@sdk-54
```

### مرحله ۲: ساخت Development Build با EAS (فقط یک بار)
```bash
npm install -g eas-cli
eas login
eas build:configure
eas build --profile development --platform android
```
- بعد از حدود ۱۰-۱۵ دقیقه، لینک دانلود یه فایل APK بهت داده می‌شه (توی ترمینال یا صفحه‌ی expo.dev)
- اون APK رو روی گوشی اندرویدت دانلود و نصب کن (شبیه یه اپ عادیه، جای Expo Go رو می‌گیره)

> برای iOS مراحل مشابهه با `--platform ios`، ولی نیاز به حساب Apple Developer داره.

### مرحله ۳: اجرا (از این به بعد، هر بار)
```bash
npx expo start --dev-client
```
یه QR کد نشون داده می‌شه؛ اینو دیگه با **اپی که خودت ساختی و نصب کردی** اسکن کن، نه با Expo Go.

> نکته: هر وقت یه پکیج نیتیو جدید اضافه شد (نه فایل جاوااسکریپت ساده)، باید مرحله‌ی ۲ رو دوباره تکرار کنی. برای تغییرات معمولی کد، همون Live Reload کار می‌کنه و نیازی به بیلد دوباره نیست.



> نکته درباره‌ی تلفظ صوتی: روی iOS اگه گوشی توی حالت سایلنت باشه صدا پخش نمی‌شه؛ سایلنت رو خاموش کن.

## نحوه‌ی اضافه کردن لغات بیشتر
لغات فعلی از یه فایل اکسل (ستون `Word` و `Persian Meaning`) گرفته و به دو درس تقسیم شدن. برای اضافه کردن لغات جدید:
1. فایل اکسل جدید رو بده تا به همون فرمت `data/words.js` تبدیلش کنم، یا
2. مستقیم یه آبجکت جدید به آرایه‌ی `WORD_LESSONS` توی `data/words.js` اضافه کن (هر آیتم یه `id`، `title` و آرایه‌ی `words` با `{id, en, fa}` داره)

## مراحل بعدی که می‌تونی اضافه کنی
- صفحه‌ی پروفایل و آمار کلی پیشرفت (جمع امتیاز هر دو بخش)
- امکان "پخش خودکار" تلفظ وقتی فلش‌کارت باز می‌شه
- گرفتن داده‌ها از یه بک‌اند واقعی به‌جای داده‌ی نمونه
- ساخت فایل نصبی (APK / IPA) با دستور `eas build` (نیاز به حساب Expo)

## ساختار پروژه
```
english-app/
├── App.js                     # نقطه ورود و ناوبری (شامل صفحه‌ی انتخاب بخش)
├── app.json                   # تنظیمات اپ (نام، آیکون و...)
├── package.json
├── data/
│   ├── lessons.js              # داده‌های درس‌های کالوکیشن
│   ├── words.js                 # داده‌های درس‌های لغات تکی
│   ├── sections.js              # رجیستری بخش‌ها (کالوکیشن/لغات) با کلید پیشرفت جدا
│   └── progress.js              # ذخیره و خواندن پیشرفت کاربر (به تفکیک بخش)
├── components/
│   └── SpeakButton.js          # دکمه‌ی تلفظ صوتی (Text to Speech) قابل استفاده مجدد
└── screens/
    ├── SectionSelectScreen.js  # انتخاب بخش (کالوکیشن یا لغات)
    ├── HomeScreen.js            # لیست درس‌های همون بخش
    ├── FlashcardScreen.js       # فلش‌کارت + دکمه‌ی تلفظ
    └── QuizScreen.js            # آزمون + دکمه‌ی تلفظ
```
